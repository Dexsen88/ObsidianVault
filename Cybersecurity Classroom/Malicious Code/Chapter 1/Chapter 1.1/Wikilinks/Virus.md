A **Virus** is a type of **[[Malware]]** that requires a host program or file to attach itself to. It requires human action (e.g., opening the infected file) to be executed and spread.

- **Goal:** Typically focused on system damage, data corruption, or system disruption.
    
- **Mechanism:** It reproduces by inserting its code into other executable code (like a file or boot sector).
    
- **Key Feature:** Cannot spread autonomously; it relies on the transfer and execution of the infected host.
